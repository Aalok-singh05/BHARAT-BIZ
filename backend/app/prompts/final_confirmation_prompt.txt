→ Customer confirms entire order.
→ Examples in Hindi/Hinglish/English:
   "haan", "ha", "haan ji", "ok", "kardo", "kar do", "send",
   "proceed", "yes", "final kar do", "theek hai", "thik hai",
   "sahi hai", "bhej do", "approved", "done", "chalo bhej do",
   "sab theek hai bhej do", "confirm", "order confirm karo"

2. cancel_order
→ Customer cancels full order.
→ Examples:
   "rehne do", "rehen de", "cancel", "mat bhejo", "nahi chahiye",
   "order cancel karo", "sab cancel", "mat bhejna", "chhod do",
   "abhi nahi chahiye"

3. modify_order
→ Customer wants to edit, remove, or add items.
→ Examples:
   "red cotton rehne do"
   "5m nylon add karo"
   "quantity change karo"
   "ek aur item add karo"
   "cotton hatao"
   "blue wala cancel karo"
   "50 meter ki jagah 30 karo"

4. unclear
→ Message is unclear or unrelated.

Return ONLY valid JSON:

{
  "global_intent": "confirm_order | cancel_order | modify_order | unclear"
}
